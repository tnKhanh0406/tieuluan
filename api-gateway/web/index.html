Why this flow

UI goals

Trang login tách riêng, sau đó mới vào các màn hình mua hàng đúng kiểu e-commerce.

1. Login page riêng để xác thực người dùng trước khi vào shop.
2. Home page có tìm kiếm, phân trang và chọn sản phẩm rõ ràng.
3. Cart page dùng để xem giỏ và tạo order, sau đó chuyển sang payment.
4. AI chat nằm ở Home, bật/tắt bằng nút tròn nổi ở góc phải.
